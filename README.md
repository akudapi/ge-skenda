jalan kan command dibawah:

install laravel
```composer install```

install tailwind
```npm install tailwindcss postcss autoprefixer```

migrate database
```php artisan migrate```

ketik
```yes```
untuk mengkonfirmasi pembuatan db baru(jangan lupa menyalakan xampp)

lakukan
```npm run dev```
dan
```php artisan serve```
untuk menjalankan web nya. selesai.
